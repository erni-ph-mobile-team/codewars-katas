### About
A collection of [Code Wars](https://www.codewars.com) katas and their solutions

#### Sharpen your skills
Challenge yourself on kata, created by the community to strengthen different skills. Master your current language of choice, or expand your understanding of a new one.

#### Train on kata
Solve the kata with your coding style right in the browser and use test cases (TDD) to check it as you progress. Retrain with new, creative, and optimized approaches.

#### Gain collaborative wisdom
Compare your solution with others after each kata for greater understanding. Discuss the kata, best practices, and innovative techniques with the community.

### Setup

- [Create an account in Code Wars](https://www.codewars.com/join).
- Update your profile in Code Wars and join the clan `ERNI Philippines`
- Install [Visual Studio Code](https://code.visualstudio.com/download) and add the extension [Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare-pack).
- Login to Live Share either using your GitHub or Microsoft account.


### Create new kata project

Run command in root directory, this will create a solution and test projects.

```bash
bash create_kata.sh <PROJECT_NAME>
```
